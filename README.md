# JCOBB DECAF B2B Landing Website

소규모 카페 대상 JCOBB DECAF B2B 납품 랜딩페이지입니다.

## 페이지 구성
- `index.html` : 메인 랜딩
- `solution.html` : 원두·레시피·POP 솔루션
- `products.html` : 디카페인 원두 & 시즌메뉴
- `inquiry.html` : 테스트빈 / 납품 문의 폼

## GitHub Pages 배포 방법
1. 새 GitHub repository 생성
2. 이 폴더 안의 모든 파일을 repository root에 업로드
3. Repository Settings > Pages
4. Source를 `Deploy from a branch`로 설정
5. Branch를 `main` / folder를 `/root`로 선택
6. 저장 후 발급되는 GitHub Pages 주소로 접속

## 실제 문의 접수 연결
현재 `inquiry.html`의 폼은 데모 동작만 합니다. 실제 접수를 받으려면 Google Forms, Tally, Formspree, Netlify Forms 등으로 연결하세요.
